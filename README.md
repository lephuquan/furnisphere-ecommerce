# FurniSphere E-commerce

## Description
An e-commerce platform for furniture shopping built with modern web technologies.

## Technologies Used
- Frontend: React.js
- Backend: Java Spring Boot
- Database: MySQL
- Authentication: JWT
- UI Framework: Material-UI/Bootstrap
- State Management: Redux
